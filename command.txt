Chuẩn, giờ bạn chạy theo đúng flow này (PowerShell, tại root repo `d:\TTTN\job_recommendation_system`).

## 0) Chuẩn bị môi trường
```powershell
$env:PYTHONUTF8='1'
$env:PYTHONIOENCODING='utf-8'
$py = if (Test-Path ".\.venv\Scripts\python.exe") { ".\.venv\Scripts\python.exe" } else { "python" }
```

## 1) Build reference mới từ job raw (skill/role/eval -> final)
```powershell
& $py deploy/scripts/reference/run_reference_pipeline.py `
  --jobs_path data/raw/jobs/topcv_all_fields_merged_2026-04-01_19-41-42.csv `
  --base_catalog data/reference/final/skill_catalog.json `
  --seed_catalog data/reference/seed/skill_catalog_seed.json `
  --staging_dir data/reference/staging `
  --final_dir data/reference/final `
  --archive_dir data/reference/archive `
  --review_report data/reference/review/validation_report.json `
  --promote `
  --include_eval_cases
```

## 2) Preprocess job raw -> artifacts matching
`run_preprocess.py` tự lấy file mới nhất trong `data/raw/jobs`. Để tránh dính `openpyxl`, ưu tiên CSV:
```powershell
(Get-Item "data/raw/jobs/topcv_all_fields_merged_2026-04-01_19-41-42.csv").LastWriteTime = Get-Date
$env:PREPROCESS_RUN_EMBEDDING='0'
$env:PREPROCESS_RUN_SECTION_EMBEDDING='0'
& $py -m src.data_processing.pipelines.run_preprocess
```

## 3) Extract CV batch
```powershell
& $py -m src.models.cv.extract_cv_batch `
  --input_dir data/raw/cv_samples/INFORMATION-TECHNOLOGY `
  --output_dir data/processed/cv_extracted `
  --aggregate_jsonl data/processed/cv_extracted/cv_extracted_dataset.jsonl `
  --aggregate_parquet data/processed/cv_extracted/cv_extracted_dataset.parquet
```

## 4) Gap analysis batch
```powershell
& $py -m src.models.cv.run_gap_batch `
  --cv_dataset data/processed/cv_extracted/cv_extracted_dataset.parquet `
  --role_profiles data/reference/final/role_profiles.json `
  --output_dir data/processed/cv_gap_reports `
  --aggregate_jsonl data/processed/cv_gap_reports/cv_gap_dataset.jsonl `
  --aggregate_parquet data/processed/cv_gap_reports/cv_gap_dataset.parquet
```

## 5) Init PostgreSQL local (migrations)
Nhớ truyền password DB local của bạn:
```powershell
.\deploy\scripts\db\init_postgres_pgvector.ps1 `
  -DbHost localhost -Port 5432 `
  -Database job_recommendation_system `
  -User postgres -Password "<YOUR_DB_PASSWORD>"
```

## 6) Load core tables vào PostgreSQL
```powershell
& $py -m src.data_processing.ingestion.load_core_tables `
  --postgres_config config/db/postgres.yaml `
  --jobs_parquet experiments/artifacts/matching/jobs_matching_ready.parquet `
  --job_skill_map experiments/artifacts/matching/job_skill_map.parquet `
  --cv_dataset data/processed/cv_extracted/cv_extracted_dataset.parquet `
  --gap_dir data/processed/cv_gap_reports
```

## 7) Chạy cơ chế chấm điểm CV mới (rubric + role benchmark + gap)
```powershell
& $py -m src.models.scoring.run_cv_scoring_batch `
  --postgres_config config/db/postgres.yaml `
```

# phiên bản mới

Nếu đi từ bước làm sạch raw data đến bước test chatbot, thứ tự chạy local nên là như này:

**1. Bật môi trường**
```powershell
cd D:\TTTN\job_recommendation_system
.\.venv\Scripts\Activate.ps1
```

**2. Bảo đảm các service nền đang chạy**
- PostgreSQL
- Neo4j
- Ollama
- Embedding service

**3. Khởi tạo schema DB**
Postgres:
```powershell
powershell -ExecutionPolicy Bypass -File deploy\scripts\db\init_postgres_pgvector.ps1 -DbHost localhost -Port 5432 -Database job_recommendation_system -User postgres -Password "<postgres_password>"
```

Neo4j:
```powershell
powershell -ExecutionPolicy Bypass -File deploy\scripts\db\init_neo4j.ps1 -CypherShellPath "<path-to-cypher-shell.bat>" -Neo4jUser neo4j -Neo4jPassword "<neo4j_password>" -Neo4jDatabase neo4j
```

**4. Làm sạch raw data, extract CV, gap, load DB, scoring, ingest, graph ETL, test API**
Nếu bạn chưa muốn chạy embedding của bước preprocess ngay:
```powershell
$env:PYTHONUTF8='1'
$env:PYTHONIOENCODING='utf-8'
.\.venv\Scripts\python.exe deploy\scripts\run_local_pipeline.py --stages preprocess_jobs extract_cv cv_gap load_core_tables cv_scoring rag_ingest graph_etl api_tests --skip-preprocess-embedding
```

**5. Chạy API server ở terminal riêng**
```powershell
$env:PYTHONUTF8='1'
$env:PYTHONIOENCODING='utf-8'
$env:EMBEDDING_SERVICE_URL='http://127.0.0.1:8081'
.\.venv\Scripts\python.exe -m uvicorn apps.api.app.server:app --host 0.0.0.0 --port 8000 --reload
```

**6. Test chatbot**
Chạy sau khi API server đã lên:
```powershell
$env:PYTHONUTF8='1'
$env:PYTHONIOENCODING='utf-8'
$env:API_BASE_URL='http://127.0.0.1:8000'
$env:CHATBOT_QUESTION='Toi la data analyst, nen chuan bi gi de len senior?'
$env:CHATBOT_TOP_K='5'
.\.venv\Scripts\python.exe deploy\scripts\run_local_pipeline.py --stages chatbot_smoke
```

**Lưu ý quan trọng**
- `chatbot_smoke` phải chạy sau khi `uvicorn` đã bật.
- `rag_ingest` cần embedding model local cache hoặc embedding config phù hợp.
- Nếu `PREPROCESS` chạy lỗi do PhoBERT chưa có cache, bạn có thể giữ `--skip-preprocess-embedding` như trên.

Nếu bạn muốn, mình có thể viết tiếp cho bạn một bản **rút gọn chỉ còn 3 lệnh chính** để bạn copy chạy nhanh mỗi lần.
